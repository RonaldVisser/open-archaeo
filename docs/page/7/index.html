Bootstrap-Correspondence-Analysis Jan 1, 0001 R-script for conducting and evaluating bootstrapped correspondence analyses.
Brainerd-Robinson-Similarity-Coefficient-and-Sampling-Error-Assessment Jan 1, 0001 Script for calculating Brainerd-Robinson coefficients based on tabular count or percent data. This script also estimates the probability that a given difference between two samples is the product of sampling error.
c14bazAAR Jan 1, 0001 R package to query different openly accessible radiocarbon date databases, allowing for basic data cleaning, calibration and merging
ca Jan 1, 0001 R Script for Seriation Using Correspondence Analysis.
caa2019_halfway_house Jan 1, 0001 This poster presents an approach that helps archaeological projects maintain an authoritative database, while retaining the flexibility and accessibility needed for ongoing research with multiple collaborators, by implementing a server-client architecture to maintain a single, consistent research database
CAinterprTools Jan 1, 0001 R package to help interpreting Correspondence Analysis' results.
calibrator Jan 1, 0001 Small command line tool for 14C calibration
carbon14 Jan 1, 0001 The goal of carbon14 is to provide a tidy interface to radiocarbon dating, to promote its inclusion in reproducible manuscripts
CAseriation Jan 1, 0001 R Package for contingency tables seriation via Correspondence Analysis.
cawd Jan 1, 0001 An R package that collects together data for the ancient world
