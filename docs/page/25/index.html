sindwem Jan 1, 0001 This R package contains exactly one function, that calculates the “sum of individual weighted means” according to a methodology that is quite common among archaeologists and was first published in 1988 by Elisabeth Fentress and Philip Perkins.
sipontomedievale Jan 1, 0001 This repo contains the code for an interactive website created to illustrate historical background, findings and data of the Siponto excavation site (Manfredonia, Italy)
sitedefinition Jan 1, 0001 The goal of sitedefinition is to see how feature densities might be used for site definition.
soiltexture Jan 1, 0001 The Soil Texture Wizard is a set of R functions designed to produce texture triangles (also called texture plots, texture diagrams, texture ternary plots), classify and transform soil textures data
sparch Jan 1, 0001 A set of extra utilities that extend the functionlaity provided by core packages for spatial data management and analysis in R
Stratibase Jan 1, 0001 Stratibase is a minimal field recording database application (currently in FileMaker FP7 format software), aimed at the simplest structure possible to perform the basic functions of a stratigraphic recording.
stratigraphr Jan 1, 0001 R package for archaeological stratigraphy and chronological sequences.
svg-pottery Jan 1, 0001 A set of best practices, how-tos and implementations for publishing pottery drawings on the Web
svg-pottery-catalogo Jan 1, 0001 A simple program that generates a catalogue of your pottery drawings ready for the Web
synthkat Jan 1, 0001 Synthetic Categories for Archaeological Vessel Comparison.
