A list of open source archaeological software and resources

archeofrag Jan 1, 0001 R package for the analysis of the fragmentation of archeological objects
arches Jan 1, 0001 Arches is a web platform for creating, managing, & visualizing geospatial data. Arches was inspired by the needs of the Cultural Heritage community, particularly the widespread need of organizations to build & manage cultural heritage inventories.
archiedb Jan 1, 0001 A light-weight database application designed to standardize and simplify data entry of archaeological or historical artifacts
ArchJobCAD Jan 1, 0001 Tasks for Excavations in AutoCAD
archSeries Jan 1, 0001 Frequency time-series tools for archaeology.
archstopmo Jan 1, 0001 An Archaeology Stop Motion Movie Festival!
archvocab Jan 1, 0001 The ArchVocab ontologies by Leif Isaksen
ARCSCore Jan 1, 0001 ARCSCore is a metadata standard for organizing and sharing digital versions of non-digital documents created during the course of an archaeological investigation
ARFloorPlanApp Jan 1, 0001 Floor plan app using augmented reality.
ark-addons Jan 1, 0001 This is a Django project to obtain statistical information from an ARK project. It is aimed to mine data from archaeological databases and create customizable visualizations, querying both text and geographical data.
