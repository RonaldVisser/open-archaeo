Sériation de matrice en présence/absence Jan 1, 0001 Excel file containing macro commands making it possible to transform a 1/0 table (presence / absence) into a graphic matrix treated by matrix permutations; it is possible to take into account the uncertain presence of a character. The instructions for use are contained in the file.
tabula Jan 1, 0001 Analysis, Seriation and Visualization of Archaeological Count Data.
teaching-rome Jan 1, 0001 A collection of texts, 3d models, and maybe other resources useful for teaching about the Roman Empire.
tfqar Jan 1, 0001 Tools for Quantitative Archaeology - in R.
The AWOL Index Jan 1, 0001 This publication systematically describes ancient-world information resources on the world-wide web. The bibliographic data presented herein has been programmatically extracted from the content of AWOL - The Ancient World Online (ISSN 2156-2253) and formatted in accordance with a structured data model
tidypaleo Jan 1, 0001 This package simplifies a few workflows when working with multi-proxy paleo-archive data
tipom Jan 1, 0001 TIPOM is based on a methodology that was developed in the 1960s by Bernardino Bagolini. The basic idea is to use the three simple dimensions of length, width and thickness of each lithic artefact to classify them in discrete groups and infer their function.
Tldating Jan 1, 0001 R package for TL dating
Tools for Quantitative Archaeology Jan 1, 0001 Keith KintighTools for Quantitative Archaeology TFQA, is a package of more than 45 PC programs developed to satisfy several unusual analytical needs of archaeologists
tosca Jan 1, 0001 Tool for Online Seriation and Correspondence Analysis.
