pollen_data_processing Jan 1, 0001 some tools in respect to pollen data processing.
poseidon-framework Jan 1, 0001 Poseidon is a framework that seeks to provide standardized ways to store and share archaeogenetic, and perhaps more broadly, molecular archaeological datasets.
Potential datasets for PhD projects Jan 1, 0001 A Google Doc with different archaeological and anthropological datasets and resources for students who need to make changes to their project due to COVID-19
pred_map_tut Jan 1, 0001 A tutorial for (archaeological) predictive mapping using r
predictSource Jan 1, 0001 A package to analyze data used to determine whether samples from multiple sources can be separated, to predict the sources of samples from unknown sources, and to evaluation the validity of those predictions. Sample data can include both quantitative and qualitative data. The package includes functions for creating an analysis file from multiple files and exploratory data analysis, as well as multivariate statistical methods to determine source separation, predict sources of unknown samples, and evaluate the validity of the predictions (principal components, classification trees, and random forests). ... ➦
profileAAR Jan 1, 0001 python qgis plugin of recexcavAAR’s archprofile
pyArchInit Jan 1, 0001 A python plugin for QGIS aimed at managing data from archaeological contexts on the GIS platform
PyXRF Jan 1, 0001 PyXRF is a python-based sophisticated fluorescence analysis package for fitting and visualizing X-ray fluorescence data
QField Jan 1, 0001 A simplified touch optimized interface for QGIS.
QGIS-scripts Jan 1, 0001 Some scripts that could be useful for QGIS users.
