ARK1 Jan 1, 0001 The ARK v1.1.2 Stable Release. ARK (The Archaeological Recording Kit) is a web-based ‘toolkit’ for the collection, storage and dissemination of archaeological data. It includes data-editing, data-creation, data-viewing and data-sharing tools, all of which are delivered using a web-based front-end.
Ark2 Jan 1, 0001 ARK (The Archaeological Recording Kit) 2.0 Development Repository. ARK (The Archaeological Recording Kit) is a web-based ‘toolkit’ for the collection, storage and dissemination of archaeological data. It includes data-editing, data-creation, data-viewing and data-sharing tools, all of which are delivered using a web-based front-end.
ArkGrid Jan 1, 0001 A QGIS plugin for local site grids.
arkhe Jan 1, 0001 A Collection of R S4 Classes that Represent Archaeological Data
ArkMatrix Jan 1, 0001 A tool for creating and manipulating Harris Matrices, part of the Archaeological Recording Kit ARK.
ArkScan Jan 1, 0001 A simple gui app to simplify scanning single context plans.
ArkSpatial Jan 1, 0001 A QGIS plugin to simplfy digitization of archaeological plans drawn using the Museum of London Single Context Recording methodology.
Arran Jan 1, 0001 A benchmark dataset for automated detection of archaeological sites on LiDAR data
arshm-release Jan 1, 0001 Experimental mobile app demoing cultural heritage documentation tools in augmented reality. Supports capturing and annotating photos in 3D space. Proof-of-concept features in the app include image anchoring and surface drawing tools.
artefact-morpho.R Jan 1, 0001 R code for converting B&W jpgs of artefact silhouettes to matrices and computing basic Elliptical Fourier statistics, including PCA and MANOVA.
